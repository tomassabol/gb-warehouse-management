# Warehouse Management

## Installation

If you don't have pnpm installed, use this command

```sh
curl -fsSL https://get.pnpm.io/install.sh | sh -
```

Install necessary dependencies

```sh
pnpm install
```

## Quick start

Start the application in development mode

```sh
pnpm run dev
```
